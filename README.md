# Customer Churn Prediction System

## 📊 Overview
Machine Learning project to predict customer churn using banking dataset.

## 🚀 Features
- Data preprocessing & feature engineering
- Random Forest model
- ROC-AUC evaluation
- Churn insights & retention strategies

## 🛠 Tech Stack
- Python
- Pandas, Scikit-learn
- Matplotlib, Seaborn

## 📈 Results
- Achieved ~84% ROC-AUC score
